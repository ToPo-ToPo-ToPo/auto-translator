# 使い方

アプリを起動すると単体ウインドウが開きます。基本は「左に入力 → 右に翻訳」だけです。

## 画面の各部

- **左ペイン（入力）:** 文章を入力すると、入力が止まったタイミングで自動翻訳します。
- **右ペイン（結果）:** 翻訳結果。右上の「コピー」でクリップボードへコピー。
  ドラッグで範囲選択して部分コピーもできます。結果は**その場で直接編集**でき、
  単語や文章を修正可能です（訳文の後編集）。編集すると「元に戻す」ボタンが現れ、
  翻訳直後の状態へ戻せます。
  - **言い換え候補:** 単語を**ダブルクリック**すると言い換え候補のメニューが出ます
    （LLMエンジン「Gemma 4（MLX）」選択時のみ）。候補を選ぶとその語を置き換えます。
- **上部バー:**
  - **エンジン選択**（左）: 翻訳エンジンを切り替え（[engines.md](engines.md)）。
  - **原文の言語**: 「自動検出」または言語を指定。
  - **⇄**: 原文／訳先の言語を入れ替え。
  - **訳先の言語**: 翻訳先を指定。
- **検出表示:** 原文を「自動検出」にした場合、判定された言語が右上に出ます。

## ログパネル（下部）

「ログ」をクリックで開閉します。起動状況・モデルのダウンロード進捗・エラーの
詳細などが表示されます。「コピー」でログ全文をコピーでき、不具合報告に使えます。
「更新」で手動更新（開いている間は自動更新されます）。

## エラー表示

エラー時は上部に「エラーがあります。詳細は下部の『ログ』を確認してください。」と
簡潔に表示されます。その文字をクリックするとログパネルが開き、詳細を確認できます。

## エンジンの切り替え

上部のエンジン選択で、Argos（既定）と Gemma 4（MLX / Apple Silicon）などを
切り替えられます。LLM（MLX）を初めて使うとき、モデルが未取得ならダウンロードの
確認ダイアログが出ます（[engines.md](engines.md)）。

## LLM モデルの自動解放

LLM（MLX / llama.cpp）は一度使うとメモリに常駐します。一定時間（既定 5 分）
翻訳を行わないと、モデルを自動でメモリから解放します。次に翻訳するときは自動で
再ロードされます。解放・再ロードの様子はログパネルで確認できます。時間は環境変数
`AUTO_TRANSLATE_IDLE_UNLOAD_SEC`（秒）で変更でき、`0` で自動解放を無効化します。
