# 使い方

アプリを起動すると単体ウインドウが開きます。基本は「左に入力 → 右に翻訳」だけです。

## 画面の各部

- **左ペイン（入力）:** 文章を入力すると、入力が止まったタイミングで自動翻訳します。
- **右ペイン（結果）:** 翻訳結果。右上の「コピー」でクリップボードへコピー。
  ドラッグで範囲選択して部分コピーもできます。結果は**その場で直接編集**でき、
  単語や文章を修正可能です（訳文の後編集）。編集すると「元に戻す」ボタンが現れ、
  翻訳直後の状態へ戻せます。
  - **言い換え候補:** 単語を**クリック**（またはフレーズをドラッグ選択）すると
    言い換え候補のメニューが出ます（LLMエンジン「Gemma 4（MLX）」選択時のみ）。
    候補を選ぶとその語が置き換わり、さらに選んだ言い換えに合うように**文全体が
    自動で調整**されます（DeepL と同様の挙動）。調整前の状態には「元に戻す」で
    戻れます。
- **上部バー:**
  - **エンジン選択**（左）: 翻訳エンジンを切り替え（[engines.md](engines.md)）。
  - **原文の言語**: 「自動検出」または言語を指定。
  - **⇄**: 原文／訳先の言語を入れ替え。
  - **訳先の言語**: 翻訳先を指定。
  - **⚙（設定）**: 設定パネルを開きます（下記）。
- **検出表示:** 原文を「自動検出」にした場合、判定された言語が右上に出ます。
  検出言語が訳先と同じとき（例: 訳先が日本語のまま日本語を入力）は、自動で
  代替言語（英語⇄日本語）へ翻訳し、訳先の選択も追従します（「検出: 日本語 →
  English に翻訳」のように表示）。

## ログパネル（下部）

「ログ」をクリックで開閉します。起動状況・モデルのダウンロード進捗・エラーの
詳細などが表示されます。「コピー」でログ全文をコピーでき、不具合報告に使えます。
「更新」で手動更新（開いている間は自動更新されます）。

## エラー表示

エラー時は上部に「エラーがあります。詳細は下部の『ログ』を確認してください。」と
簡潔に表示されます。その文字をクリックするとログパネルが開き、詳細を確認できます。

## エンジンの切り替え

上部のエンジン選択で、Argos（既定）と Gemma 4（MLX / Apple Silicon）などを
切り替えられます。LLM（MLX）を初めて使うとき、モデルが未取得ならダウンロードの
確認ダイアログが出ます（[engines.md](engines.md)）。

## 設定パネル（⚙）

上部の **⚙** ボタンで設定パネルを開きます。変更は「保存」で反映・保存され、次回
起動時にも引き継がれます（`~/.config/auto-translator/settings.json`）。「既定に戻す」
で初期値に戻せます（保存で確定）。

- **表示:** テーマ（システム/ダーク/ライト）、入力・訳文の文字サイズ。
- **翻訳の動作:**
  - **入力中に自動翻訳:** オフにすると、入力欄からフォーカスを外したとき、または
    **⌘/Ctrl+Enter** で翻訳します。
  - **自動翻訳までの待ち時間:** 入力が止まってから翻訳するまでの待ち時間（ms）。
- **LLM（Gemma）パラメータ**（MLX / llama.cpp 使用時に反映。Argos では無効）:
  - **生成温度（temperature）:** 低いほど直訳的で安定、高いほど多様な訳になります。
  - **最大生成トークン:** `自動`（入力長に応じて調整）または上限を指定。
  - **未使用モデルの自動解放:** 下記。`無効`で自動解放しません。

## LLM モデルの自動解放

LLM（MLX / llama.cpp）は一度使うとメモリに常駐します。一定時間（既定 5 分）
翻訳を行わないと、モデルを自動でメモリから解放します。次に翻訳するときは自動で
再ロードされます。解放・再ロードの様子はログパネルで確認できます。

時間は設定パネルの「未使用モデルの自動解放」で変更でき、`無効`で自動解放を止めます。
環境変数 `AUTO_TRANSLATE_IDLE_UNLOAD_SEC`（秒）を設定した場合はそちらが優先されます
（`0` で無効）。
